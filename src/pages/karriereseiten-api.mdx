Eine API-Integration ist der komplizierteste Weg, aber er bietet Ihnen unbegrenzte Gestaltungsmöglichkeiten. 
Sie basiert auf REST.
Funktionen:
- Job Export -> REST API / XML Feed zum Lesen von Stellenanzeigen
- Kandidatenimport ->REST API zum Erstellen von Bewerbern, Bewerbungen, Dateianhängen undweiteren Stammdaten
- Status-Synchronisierung
- flexibel erweiterbar
- Optional: E-Mail-Benachrichtigung, Posting zu Microsoft Teams, Slack, Stellenanzeigenposting zu 100+ Plattformen und Suchmaschinen, Onboarding, etc.
- DSGVO-konforme Löschung & Self-Service-Feature

Hinweis: APIs benötigen fundierte Programmierkenntnisse! Als API-Benutzer verwalten Sie die Businesslogik Ihrer Karriereseite komplett selbst.

Für eine technische Beschreibung der Karriereseiten API kontaktieren Sie bitte support@inriva.com 

<img src="/images/faqs/custom+api.png" alt="Karriereseiten-API" style="width: 100%"/>

Hinweis: Wenn Sie die Karriereseiten API nutzen um Jobs auf Ihre eigene Website zu veröffentlichen, wird eine deaktivierte Standard-Karriereportal-Plattform für Sie veröffentlicht (ihrkarriereportal.inriva.com). Bitte kontaktieren Sie support@inriva.com wenn Sie diese aktivieren oder ändern möchten.
