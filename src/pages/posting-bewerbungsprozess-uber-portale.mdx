featured: false
---

Die meisten Stellenportale unterstützen eine Verlinkung der Bewerbungsfunktion direkt zur Karriereseite des Unternehmens. Bei jedem Posting wird diese Bewerbungs-URL mitgegeben.
So ist für die meisten Portale der Bewerbungsprozess komplett durchgängig – bspw. schön zu sehen auf Indeed https://de.indeed.com/viewjob?jk=74784556ccccd673&tk=1c6ro7uam16vo0hk&from=serp&vjs=3 .

Es kann allerdings nicht ausgeschlossen werden dass sich ein Bewerber per Mail bei Ihnen bewirbt oder eine Plattform auf eine Bewerbung auf den eigenen Bewerbungsweg besteht.

Hier kann die Bewerbung entweder akzeptiert oder mit Hinweis auf das Portal (automatisiert oder manuell) zurückgewiesen werden.
